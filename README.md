# Task Dashboard

## Description

This project is a task management dashboard built with Next.js and Tailwind CSS. Users can add, complete, delete, filter, and save tasks using localStorage.

## Features

- Add new tasks
- Toggle task completion
- Delete individual tasks
- Filter by All, Active, and Done
- Live task statistics
- Clear completed tasks
- Persist tasks after browser refresh using localStorage

## Installation

1. Clone the repository.
2. Run:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open http://localhost:3000
